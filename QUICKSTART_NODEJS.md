# Быстрый старт - Node.js версия

## Установка и запуск

1. **Установите зависимости:**
```bash
npm install
```

2. **Инициализируйте базу данных:**
```bash
npm run init-db
```

3. **Запустите сервер:**
```bash
npm start
```

Или используйте скрипт:
```bash
./run_nodejs.sh
```

## Доступ

- Приложение: http://localhost:8000
- API: http://localhost:8000/api

## Учетные данные

- Username: `admin`
- Password: `admin123`

## Разработка

Для разработки с автоперезагрузкой:
```bash
npm run dev
```

## Миграция с Python версии

База данных SQLite полностью совместима. Вы можете использовать существующую базу данных `chat_video.db` без изменений.

## Переменные окружения

Создайте файл `.env` (опционально):
```env
SECRET_KEY=your-secret-key
DATABASE_URL=sqlite:///./chat_video.db
PORT=8000
```
